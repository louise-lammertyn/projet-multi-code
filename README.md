# Modelisation of Okayama's quadrupole
This GitHub repository contains all the necessary codes and documents to implement the modelisation of Okayama's quadrupole *[S.OkayamaetH.Kawakatsu, Potential distributionand focal pro
pertiesofelectrostaticquadrupole lenses ,JournalofPhysicsE:Scienti cInstruments,vol.11,no3,
p.211,1978]*.\
\
In the folder "Quadrupole_model", you can find all the python programs that modelise the quadrupole. The users' files are "Potential_extraction", that extracts the potential of the volume using BEMPP and "Data_exploitation.ipynb" that uses the created data and calculates the trajectories of the electron beams and the aberrations of the system. The files exported through these programs are saved in the folder "Files".\
More details about the calculation that has been done can be found in the document "Rapport_final.pdf" in the folder "Documentation".\
